# Login

The `Login` class is the main screen manager for the Universal Login **Login** experience.  
It provides methods for:

- Username/password login  
- Social / federated login  
- Country code selection for phone inputs  
- Reading tenant, client, user, and transaction context  
- Retrieving transaction errors  
- Reading active login identifiers  

Defined in:  
`src/screens/login/index.ts:26`  
https://github.com/auth0/universal-login/blob/5c054686de6099ea69493d68134be65604477bb6/packages/auth0-acul-js/src/screens/login/index.ts#L26

<RequestExample>
```ts
import Login from "@auth0/auth0-acul-js/login";

const loginManager = new Login();

// Username/password login
await loginManager.login({
  username: "testUser",
  password: "testPassword",
});

// Social / federated login
await loginManager.federatedLogin({
  connection: "google-oauth2",
});

// Optional: open country code picker for phone input
await loginManager.pickCountryCode();

// Read active login identifiers
const identifiers = loginManager.getLoginIdentifiers();

// Inspect transaction errors
const errors = loginManager.getErrors();
```
</RequestExample>
